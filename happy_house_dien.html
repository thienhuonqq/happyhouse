🟢 Kết nối với Google Sheets / Excel
Cách 1 – Copy-paste thủ công (đơn giản nhất):
  1. Nhấn Xuất CSV để tải file dữ liệu về máy
  2. Mở Google Sheets → File → Import → Upload → chọn file .csv vừa tải
  3. Chọn "Replace spreadsheet" hoặc "Insert new sheet" tuỳ ý
Cách 2 – Tự động với file Excel có sẵn "Happy House admin.xlsx":
  1. Mở file Happy House admin.xlsx
  2. Tạo sheet mới tên "Data" và paste dữ liệu CSV vào đó
  3. Dùng =VLOOKUP() trong các sheet tháng để kéo số liệu tự động
💡 Tự động hoàn toàn: Dùng Zapier hoặc Make.com (miễn phí) để mỗi lần nhấn Lưu sẽ tự ghi vào Google Sheets. Nhắn mình cần hỗ trợ setup nếu cần nhé!
